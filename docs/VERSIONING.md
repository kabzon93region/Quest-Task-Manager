# Схема версий QTaskMgr

Формат: **`X.Y.Z`**

| Часть | Имя | Диапазон | Когда менять |
|-------|-----|----------|--------------|
| **X** | Эволюция | 1… | Только по указанию автора. При смене эволюции весь функционал становится **базовым**, **Y → 0**. |
| **Y** | Функции | 0…99 | +1 за каждую **новую** возможность в текущей эволюции. При Y=99 следующая функция повышает X, Y → 0. |
| **Z** | Исправления | 0… | +1 за релиз с исправлениями **без** новой функции. |

**versionCode** (Android): `X × 10000 + Y × 100 + Z` (пример: `1.2.7` → `10207`).
