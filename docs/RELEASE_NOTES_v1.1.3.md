# Quest Task Manager v1.1.3

## Что нового в v1.1.3

- **Настройки Android:** открывается главный экран (Приложения, Система, Спец. возможности), а не сразу режим разработчика.
- **Запущенные:** больше процессов — `ps` + `dumpsys activity processes` + summary `dumpsys meminfo`.
- **Тап по плашке** в «Запущенные» → страница приложения в Android Settings (открыть / закрыть / разрешения / память).
- **Закрытие:** усилено (`force-stop` → `kill` → `kill -9`); toast показывает защищённые и не закрытые apps.
