# Quest Task Manager v1.4.0

**Поколение 1.2–1.4** — эволюция 1.  
**v1.4.0** — накопительный релиз: всё новое и исправленное с **v1.2.14** без публикации промежуточных тегов (v1.3.x).  
Опубликован на GitHub как последний major-релиз поколения 1.4; следующий патч — **v1.4.1** (DonationAlerts).

APK: `QTaskMgr-v1.4.0-release.apk`

---

## Новые возможности

### Фильтр «Демоны» (вкладки «Запущенные» и «Приложения»)

- Отдельный фильтр для нативных процессов (`media.*`, `android.*`, `hidl.*` и т.д.).
- В фильтре **«Все»** демоны **не показываются** — только в «Демоны».
- Закрытие: `am force-stop` для пакетов, `kill -9` по PID для нативных процессов.
- Политики фона — если применимы к процессу.

### Поиск приложений

- Поле **Поиск** (до 7 символов) справа от фильтров категорий на обеих вкладках списков.
- Фильтрация по **названию** и **имени пакета** при каждом вводе или удалении символа.
- Пустое поле — без поискового фильтра (действует только категория: Все / Пользовательские / Системные / Демоны).

### Настройки

- Отображается **версия** приложения.
- Кнопка **«Страница на GitHub»** — [github.com/kabzon93region/quest-task-manager](https://github.com/kabzon93region/quest-task-manager).

## Закрытие приложений

- **«По правилам»**, **«Закрыть все»** и очистка по **уведомлению** проверяют **весь список** приложений, **независимо** от активного фильтра отображения (Пользовательские / Системные и т.д.).
- Можно закрывать **защищённые** системные пакеты (VrShell, Meta и др.) — исключение только само QTaskMgr.
- Очистка по уведомлению: та же логика, что «По правилам»; исправлен узкий список кандидатов (раньше многие apps не попадали в выборку).

---

## Уведомление

- После очистки корректно показывается **«Закрыто: N»** (раньше число могло не отображаться).
- Формат: `2.1/5.8 GB • Закрыто: 3`.

---

## Интерфейс и лаунчер

- В лаунчере Quest отображается **QTaskMgr**, а не имя пакета `com.quest3.taskmanager`.
- Добавлены иконки приложения (adaptive + PNG).
- Индикатор загрузки списка не пропадает при смене фильтра во время обновления (счётчик параллельных загрузок).
- Полоска прогресса перенесена над списком — всегда видна при обновлении.

---

## Установка

Требуется **Shizuku** на Quest. Рекомендуется обновление с **v1.2.14** или любой более ранней версии 1.x.

```powershell
adb install -r dist\QTaskMgr-v1.4.0-release.apk
```
