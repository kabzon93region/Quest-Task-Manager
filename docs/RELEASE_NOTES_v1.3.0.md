# Quest Task Manager v1.3.0

## Что нового в v1.3.0

- Фильтр **«Демоны»** на вкладках «Запущенные» и «Приложения» — нативные процессы (`media.*`, `android.*`, `hidl.*` и т.д.).
- В фильтре **«Все»** демоны не показываются.
- Закрытие демонов: `am force-stop` для пакетов, `kill -9` по PID для нативных процессов.
- Политики фона — если применимы к процессу (есть пакет в системе).
