# Quest Task Manager v1.1.0

## Что нового в v1.1.0

Первый релиз линейки v1.1:

- Вкладки **Запущенные**, **Приложения**, **Настройки**
- Закрытие процессов (ps + meminfo), kill all / selected / по правилам
- Переключатели фона и фоновых данных (appops / netpolicy)
- Диск и RAM отдельно в карточках приложений
- Фоновая очистка по уведомлению (`CleanupForegroundService`)
- Кнопка **Настройки Android**, лог на устройстве
- Не требует No More Background и QTKiller
